
# アクションフェイズでできること									
									
## 基本

1ターンごとに次のうち1つを行える								
							
+ ナラティブの取得 : 取得したいナラティブを1つ選ぶ、ランクを難易度に情報の判定に成功するとそのナラティブを1つ取得できる
+ アクションフェイズに使用できるナラティブを1つ使用すること
+ アクションフェイズにできるスキルを1つ使用すること
+ ナラティブを他のキャラクターに受け渡す、引き継ぐ、交換する

## 行動の順番

運の現在値が最も高いキャラクターから行動する

## 判定方法

破壊、情報、回避などの能力値をもとに、その数だけ6面ダイスを振ります
そのときの好きな出目を結果として利用できます

また、1の出目が2つ以上出た場合ファンブルとして扱いファンブル表を振ります
ファンブルしたからといって、必ず失敗とはなりません、ファンブル表の結果に従います
これは、他の出目がでていても1の出目が2つ以上でていたらファンブルとしてあつかいます。

## ファンブル表

+ [02] 判定を失敗として扱う、ナラティブロスト5
+ [03] 判定を失敗として扱う、2DD点のダメージを受ける
+ [04] 判定を失敗として扱う、1D6点のダメージを受ける
+ [05] 3点のダメージを受ける
+ [06] ナラティブロスト1
+ [07] 1D6点のダメージを受ける
+ [08] 判定を失敗として扱う
+ [09] ナラティブロスト3
+ [10] 判定を失敗として扱う、ナラティブロスト1
+ [11] 2DD点のダメージを受ける
+ [12] ヒール2
