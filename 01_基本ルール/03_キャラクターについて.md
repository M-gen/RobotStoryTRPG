# キャラクターについて

+ 基本能力
    + 情報 情報の収集や、ナラティブの取得に必要になる力
    + 破壊 破壊活動にまつわる力
    + 回避 逃げたり移動したりする力
+ 資源能力
    + 運 このゲームの資源であり0となると死亡します、ダメージなどもこの値で受ける
+ スキル いくつか保持することができ、決まったタイミングで使用できます。
+ ナラティブポイント ナラティブを保持し続ける時に必要になります。

## 作り方

次のキャラクターテーマから1つ決めて、1d6などはダイスを振って決めていきます

### 機械に詳しい

    情報 : 3
    破壊 : 1d6
    回避 : 3
    運   : 17+3d6
    スキル : 2コ
    ナラティブポイント : 2+2d6

## キャラクターの死亡について

運が0になったキャラクターは死亡とし、次回以降のセッションで使用できなくなります。

死亡したセッションでも、運や能力値を使用しないスキルだけが使用でき、効果を発動できますので、最後まで参加することができます。

たとえば、「ブラザー」は使えませんが、「連携」や「ムードメーカー」は使用できます。


## その他

### Q.セッションまでにできることは？

A.ナラティブの取得などはセッションはじまるまでできません。

### Q.キャラクター作成のダイスの振り直しについて
A.各卓で判断して下さい。目安として『1通りのダイスを振った結果を1セットとして、最大3セットし、その中から1つ選ぶ』ということも可能です。
