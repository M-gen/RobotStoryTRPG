
# ガンダム二次創作TRPG - えむげん式0.1

## 概要

本ゲームは、ガンダムのTVシリーズのような物語が量産されるようなTRPGを目指しています。

## 参加者

遊ぶとき、監督1人と複数人のプレーヤーに分かれて遊ぶ

### 監督
セッションを進行する役で、セッション用のシナリオを準備し、それにもとづいて脅威などを運用する
あらゆるルールについて手を加える権限を持つ
プレーヤーとともにキャラクターを作成しセッションに参加してもよい。

### プレーヤー
キャラクターを作成し、セッションに挑み、セッションでの生き残りを第1目標に、セッションのクリアーを第2目標に、キャラクター表現を第3目標に
ルールに基づいて各自が作ったキャラクターを運用する。

## セッションの流れ

セッション全体を次のように構造化します

    (1) ブック1
    (2) ブック2
    (3) ブック3
    (4) ブック4(ラストブック)
    (5) エピローグ

まず、(1)～(4)でブックの処理を繰り返します。ブック4はラストブックとなるため、ここで脅威が残っていたら敗北となります。

ブックとは、物語を脅威への対処を中心に考えた区間をあらわしています。

ブックでは、

    (1) スタートフェイズ
    (2) アクションフェイズ
    (3) エンドフェイズ

と順に処理します

スタートフェイズでは、
GMから脅威やナラティブをシナリオごとのブック[N]ランダム表から決めます。
このとき、この脅威に対処していられる時間、アクションフェイズで活動できるターン数も確定します。
ここでは出現する脅威が確定し、場合によってはナラティブを取得できます。

エンドフェイズでは
脅威が失敗だったとき、失敗ペナルティが発動し、成功していたら成功ボーナスが発動します。
各キャラクターのナラティブのランクの合計がナラティブポイント以下になるようナラティブを調整します。このとき、プレーヤーキャラクター間で自由にナラティブを交換できます。

最終的に、エピローグまで生き残りなるべく脅威を対処していくのが本ゲームの基本構造です。

## エピローグ

セッション終了時は、全員が経験点を1点獲得ます。
さらに、ラストブックで脅威が全部対処できていた場合はミッション成功として、経験点を追加で1点獲得します。

### ナラティブの引継ぎ

ナラティブポイントの半分だけ（端数切り上げ）、各キャラクターはナラティブを次のセッションに引き継げます。

### 経験点の使用と成長

次のセッションまでに、キャラクターを成長させることができます。

1点を消費して1d6の運の回復します。元の最大値は超えません。

2点を消費して1点のナラティブポイントを増加させられます。

4点を消費して1点、情報か破壊か回避の能力値を1つ最大6まで上昇させることができます。

2点を消費して1つスキルを増やすことができます。

+ 基本能力

    + 破壊 破壊活動にまつわる力

## キャラクターの死亡について

運が0になったキャラクターは死亡とし、次回以降のセッションで使用できなくなります。

死亡したセッションでも、運や能力値を使用しないスキルだけが使用でき、効果を発動できますので、最後まで参加することができます。

たとえば、「ブラザー」は使えませんが、「連携」や「ムードメーカー」は使用できます。

## 特殊用語

### [数字A]D[数字B]

    数字Bで指定されたダイスを、数字Aでの数だけふることを表現する
    「1d6」なら「6面ダイスを1コふる」となる

### ブリーフィング[数字A]	

	数字Aの数だけ、PC全員のすべての判定の達成値を増加する
	この効果は重複しない。数字Aが異なるばあいは大きいほうを優先する。
		
###	ナラティブロスト[数字A]	
		
    ナラティブを失う効果
    ナラティブのランクが合計して指定された数字A以上になる分だけ、ナラティブを失う
    
###	ナラティブハック[数字A]	
		
    ナラティブを取得する
    数字Aまでのランクのナラティブを1つ入手する
    
###	ミニボム[数字A]	

	[【残りの運】×(数字A)D6]点のダメージを任意のPC1体に与える
		
		
###	バースト[数字A]	

	[【残りの運】×(数字A)D6]点のダメージを全てのPCに与える
		
### ヒール[数字A]	

	数字Aの人数だけキャラクターを選択する、その運を1D6点回復する

### 対脅威:[属性A][数字A]

    それを使用したときに、属性Aと対応する脅威にダメージを与えることができる
    脅威の難易度を目標に属性Aで判定する。
    成功した場合、脅威の運に数字Aのダメージを与える。

## スキル

### ゾロゾロボーナス
    判定のダイスで、同じで目が出た時、それの出目を結果にするなら、出た目の個数とおなじだけ結果をプラスしてよい
    例えば [2, 2 , 2,  1] という結果であれば、2を採用した場合は、
    そこに2が3つでているので 2 + 3 で 5 とあつかってよい

### 悪運
    失敗したブックのエンドフェイズで、[1D6]点の運を回復する

### コンスタント
    ブックのエンドフェイズで、1点の運を回復する

### 安全マージン
    「判定のダイスをN個減らす代わりにその結果を+Nする」ことを選択してよい

### ブラザー
    他のキャラクターがダメージを受ける時、2d6点まで代わりにダメージを受ける

### 徹夜
    1つのアクションフェイズで、2点のダメージを受けることで、アクションフェイズで行う行動を1つ増やすことができる

### コメディ
    1つのブックに3回まで、判定に失敗したとき1点の運を回復する

### 先読み
    キャラクター1体を対象にする、このアクションフェイズでの対象の判定を+1する

### 連携
    行動を消費せずにナラティブを1つ交換、受け取り、受け渡すことができる

### ギャンブラー
    1d6を振り奇数であれば3回行動できる

### オーバードライブ
    ナラティブによる対脅威で与えるダメージを+2するかわりにそのナラティブを破壊してもよい

### リサイクル
    ナラティブを1つ破棄することで自信の判定の達成値を+1する

### ガッツ
    運を1点消費する代わりに、判定の達成値を+1してもよい

### ムードメーカー
    ブリーフィングの効果を+1する

## アクションフェイズでできること									
									
### 基本

1ターンごとに次のうち1つを行える								
							
+ ナラティブの取得 : 取得したいナラティブを1つ選ぶ、ランクを難易度に情報の判定に成功するとそのナラティブを1つ取得できる
+ アクションフェイズに使用できるナラティブを1つ使用すること
+ アクションフェイズにできるスキルを1つ使用すること
+ ナラティブを他のキャラクターに受け渡す、引き継ぐ、交換する

### 行動の順番

運の現在値が最も高いキャラクターから行動する

## 判定方法

破壊、情報、回避などの能力値をもとに、その数だけ6面ダイスを振ります
そのときの好きな出目を結果として利用できます

また、1の出目が2つ以上出た場合ファンブルとして扱いファンブル表を振ります
ファンブルしたからといって、必ず失敗とはなりません、ファンブル表の結果に従います
これは、他の出目がでていても1の出目が2つ以上でていたらファンブルとしてあつかいます。

### ファンブル表

+ [02] 判定を失敗として扱う、ナラティブロスト5
+ [03] 判定を失敗として扱う、2d6点のダメージを受ける
+ [04] 判定を失敗として扱う、バースト2
+ [05] バースト1
+ [06] ナラティブロスト1
+ [07] 1d6点のダメージを受ける
+ [08] 判定を失敗として扱う
+ [09] ナラティブロスト3
+ [10] 判定を失敗として扱う、ナラティブロスト1
+ [11] 2d6点のダメージを受ける
+ [12] ヒール2
